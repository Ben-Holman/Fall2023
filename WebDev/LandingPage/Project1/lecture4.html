Internet Protocols:

Internet Protocols are rules used for communication between one or more machines.TCP/IP is an example of a protocol, but there are many others. these protocols are written into every operating systems and are uniform so machines running different operating systems can still communicate with each other. The protocols are designed into a stack. The stack can very between 4 layers and 8 layers, but it is all the preference of the operating system designer. The layers are in order as follows:

  • Link Layer
  • Internet Layer
  • Transport Layer
  • Application Layer

Domain Name System:

In the days of the ARPANET users didn't want to type in a specific IP address to access pages. So researches using the ARPANET created Domains that would like to an IP address. As the ARPANET grew into the larger Internet, domains needed to be catagorized for quick and easy access. Enter Domain Name System(DNS). A DNS is a specific web server that takes incoming webpage requests and sends the correct webpage content from the accessed IP address back to the requesting user. For example: A user enters in www.google.com. that web request goes to a DNS server, that server searches the corresponding IP address, in this case being: 142.250.191.46, and sends that webpage content back to the user and google.com appears on your screen. The DNS can be broken up into different levels that have different jobs and speed up searching and retrieving processes.

URLs:

Uniform Resource Locators(URLs) allows the user to send a request for specific content from a web server. The URL's consist of he protocols to connect to the server and the Domain name of the webpage trying to be accessed. URL's can be modified to access specific parts of webpages instead of going to the Landing Page of the website. For example: you can go to YouTube with "youtube.com" as the URL and be directed to the home page. Or, you can be more specific and request "youtube.com/watch?v=[VIDEO_ID]" and get the video you're wanting to watch directly from the URL.
