The Internet: A Complex Ecosystem

In Lecture 2, we learn about the history of the internet from it's Cold War beginnings to today's complex web of services and web applications. The ARPANET, created in the 1960's gave rise to the early internet through the use of packet switching and protocols. Then Sr. Tim Berners-Lee, in 1992, sets the standards for the World Wide Web(WWW). The Uniform Resource Locator(URL) gives each webpage a unique ID to be searched out on the web. The Hypertext Transfer Protocol (HTTP) is used for request handling from client to server and vice versa. The Web Server Software is software that is designed specifically to handle HTTP requests. The Hypertext Markup Language(HTML) is the programming language used to create and publish pages and documents on the web. This web page was created using HTML! Finally, you need something to interpret HTTP, URLs and HTML, and that is the browser. Edge, FireFox, Chrome, and Safari are all different browsers.

Web 2.0

The introduction of Javascript gave web developers the tools to create more interactive webpages and through that, create a more user-driven experience. Web 2.0 focused heavily on browser use and relying less on the server to deliever dynamic web content. This broswer-focused development meant that web developers needed to learn Javascript.
